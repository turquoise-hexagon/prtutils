-------
prtsize
-------

find size of a installed package
================================

:date: March 2020
:version: 0.0
:manual section: 1
:manual group: General Commands Manual

synopsis
--------
prtsize `<package>`

description
-----------
prtsize returns the total size of the files that were installed with a package, in human-readable format

example
-------
::

    $ prtsize neovim
    16.6M
